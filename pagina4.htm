PÁGINAS WEB 
    


•CONCEPTO:
Una página web, página electrónica o página digital es un documento de carácter multimediático, es decir, capaz de incluir audio, video, texto y sus combinaciones, que presenta su información de manera organizada y se encuentra alojado en internet. Se trata del formato básico de contenidos en la Word Wide Web (WWW).

•TIPOS:
-Estática: Una página web estática es un conjunto de páginas web cuyo contenido es el mismo cada vez que los usuarios acceden a ella. Comparado con un sitio web dinámico, es más rápido y fácil de crear, pero tiene una interactividad limitada.
 -Dinámica: Una página web dinámica muestra una variedad de contenido para diversos usuarios.

•CONSTRUCCIÓN:
Lenguajes de programación web
-HTML: Es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web.
 -ASP: es la tecnología desarrollada por Microsoft para la creación de páginas dinámicas del servidor.
-PHP: es un lenguaje de programación de propósito general, especialmente orientado al desarrollo web.
